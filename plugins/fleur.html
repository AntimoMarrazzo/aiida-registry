AiiDA plugin package "aiida-fleur"

< back to the registry index

General information

Current state: development Adds new functionality, not yet ready for production. Testing welcome!

Short description: Python FLEUR simulation package containing an AiiDA Plugin for running the FLEUR-code and its input generator. Plus some workflows and utility

How to install: pip install aiida-fleur

Source code: Go to the source code repository

Documentation: Go to plugin documentation

Detailed information

Author(s): JuDFTteam

Contact: j.broeder@fz-juelich.de

How to use from python: import aiida_fleur

Most recent version: 1.1.0

Compatible with aiida-core: >=1.0.1,<2.0.0
